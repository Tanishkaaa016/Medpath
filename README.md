# 🩺 MedPath – Smart Symptom-to-Diagnosis Web App

MedPath is a machine learning-powered web application that predicts possible diseases based on user-input symptoms. Built using Flask and an Artificial Neural Network (ANN), it provides quick and interactive medical insights for educational purposes.

---

## 🚀 Features

* 🔍 Input multiple symptoms
* 🧠 Predict diseases using ANN / ML logic
* 📊 Ranked results with top diagnosis
* 🌐 Clean and interactive web interface
* ⚡ Fast API-based backend with Flask

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Flask (Python)
* **Machine Learning:** ANN / TensorFlow (if used)
* **Deployment:** Render
* **Other:** Flask-CORS, NumPy

## ⚙️ Installation (Local Setup)

```bash
git clone https://github.com/YOUR_USERNAME/medpath.git
cd medpath

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
python app.py
```
